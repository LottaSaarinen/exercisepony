useLocalStorage(key, defaultState)

React Hook, tallentaa ja hakee tietoja `localStorage`-muistista. hook mahdollistaa tietojen tallentamisen paikalliseen tallennustilaan ja tietojen palauttamisen käyttöliittymän uudelleenrenderöityessä.
Parameters:
Name Type Description
key string `localStorage`-avain, käytetään tietojen tallentamiseen ja hakemiseen.
defaultState any Alkuarvo palautetaan, jos ei ole aiemmin tallennettua arvoa.
Source:
  • useLocalStorage.js, line 34
Returns:
Example
import useLocalStorage from './useLocalStorage.js'

const ExampleComponent = () => {
  //`useLocalStorage`-hook. määritellään avain ja alkuarvo.
  const [value, setValue, resetValue] = useLocalStorage('myKey', 'defaultValue');

  return (
    <div>
      <p>Tallennettu arvo: {value}</p>
      <button onClick={() => setValue('newValue')}>Päivitä arvo</button>
      <button onClick={resetValue}>Nollaa arvo</button>
    </div>
  );
};
